void OpenCVForUnity.XimgprocModule.GraphSegmentation.processImage ( Mat  src,
Mat  dst 
)

Segment an image and store output in dst param src The input image. Any number of channel (1 (Eg: Gray), 3 (Eg: RGB), 4 (Eg: RGB-D)) can be provided param dst The output segmentation. It's a CV_32SC1 Mat with the same number of cols and rows as input image, with an unique, sequential, id for each pixel.
