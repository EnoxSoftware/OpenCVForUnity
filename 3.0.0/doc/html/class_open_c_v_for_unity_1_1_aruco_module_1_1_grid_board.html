Detailed Description

Planar board with grid arrangement of markers More common type of board. All markers are placed in the same plane in a grid arrangement. The board can be drawn using drawPlanarBoard() function (SEE: drawPlanarBoard)

Draw a GridBoard

param outSize size of the output image in pixels. param img output image with the board. The size of this image will be outSize and the board will be on the center, keeping the board proportions. param marginSize minimum margins (in pixels) of the board in the output image param borderBits width of the marker borders.

This function return the image of the GridBoard, ready to be printed.
