void OpenCVForUnity.XimgprocModule.FastLineDetector.detect ( Mat  image,
Mat  lines 
)

Finds lines in the input image. This is the output of the default parameters of the algorithm on the above shown image.

param image A grayscale (CV_8UC1) input image. If only a roi needs to be selected, use: fld_ptr-&gt;detect(image(roi), lines, ...); lines += Scalar(roi.x, roi.y, roi.x, roi.y); param lines A vector of Vec4f elements specifying the beginning and ending point of a line. Where Vec4f is (x1, y1, x2, y2), point 1 is the start, point 2 - end. Returned lines are directed so that the brighter side is on their left.
