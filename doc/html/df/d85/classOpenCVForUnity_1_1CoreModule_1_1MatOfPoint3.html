Detailed Description

A specialized Mat class for storing 3D points with 32-bit integer coordinates (CV_32SC3).

This class represents a standard Mat object of type CV_32SC3, where each element corresponds to a 3D point with 32-bit integer x, y, and z coordinates. The toArray and toList methods allow easy access to the data as an array or list of Point3 objects, making it useful for representing 3D points or shapes in OpenCV.
